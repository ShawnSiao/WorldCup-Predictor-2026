# 预测配图

[English](README.md) | [简体中文](README.zh-CN.md)

本目录存放由 `$imagegen` 生成的比赛预测和每日报告分享配图。

图片必须是真实位图文件，例如 PNG、JPG、JPEG 或 WebP。不得使用 SVG、HTML、canvas、Mermaid、纯图表或脚本生成的替代图片。

使用以下指令形式：

```text
$imagegen: 生成【社交平台赛事预测配图】，16:9 横版，真实位图图片，用于抖音、小红书、微博和微信分享；中文文档配图的主要比赛信息必须使用简体中文，可在画面合适位置保留英文队名/赛事信息作为辅助文字；不要生成 SVG，不要生成 HTML，不要生成代码图，不要生成线框图，不要使用官方 FIFA 标志或水印。
```

`data/predictions.json` 中的每条预测记录都必须包含本目录下的 `image_file` 路径。英文预测页、简体中文预测页和对应日报都应嵌入同一张图片，便于 GitHub 读者和社交平台编辑直接复用。
