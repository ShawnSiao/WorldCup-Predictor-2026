## 仓库结构

- `data/` 存储赛程、球队、场馆、排名、预测、结果和复盘索引。
- `predictions/` 存储赛前预测文件。
- `reviews/` 存储官方结果确认后的赛后复盘。
- `reports/daily/` 存储每日跟踪报告。
- `docs/` 存储方法论、来源和数据结构文档。

比赛生命周期：

```text
scheduled -> predicted -> live -> final -> reviewed
```
