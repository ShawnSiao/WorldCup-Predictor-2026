本仓库用于追踪 2026 年 FIFA 世界杯赛程，记录每场比赛的赛前预测，并将预测结果与最终赛果进行对比。

当前处于项目初始化阶段。赛程数据、预测流程和赛果追踪工具会在后续提交中补充。

## 追踪内容

- 按小组、轮次、场地和开球时间整理赛事赛程。
- 记录每场比赛的预测胜者、预测比分、信心等级和备注。
- 保存最终比分、胜者、点球结果和预测得分。
- 保留历史预测记录，便于后续分析。

## 计划数据模型

| 实体 | 用途 |
| --- | --- |
| `teams` | 参赛球队与小组分配。 |
| `matches` | 开球时间、比赛场地、小组或淘汰赛轮次、参赛双方和比分。 |
| `predictions` | 预测胜者、预测比分、信心等级和备注。 |
| `results` | 最终比分、胜者、点球结果和预测得分。 |
