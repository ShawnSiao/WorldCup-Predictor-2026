# 预测校准

[English](prediction-calibration.md) | [简体中文](prediction-calibration.zh-CN.md) | [方法说明](methodology.zh-CN.md)

本文定义证据覆盖检查之后、发布胜平负概率、比分、信心等级和平台文案之前使用的校准层。

## 数据缺口补齐

数据缺口分三层处理：

| 层级 | 缺口类型 | 处理方式 |
| --- | --- | --- |
| 自动补齐 | 赛程、场地、赛果、排名、名单发布状态 | 从 FIFA、官方足协比赛页和可靠赛程/赛果交叉来源刷新结构化数据。 |
| 半自动补齐 | 天气、草皮/屋顶、公开市场变化、专家预览 | 在 T-72h、T-24h、T-6h 抓取或手工记录可靠来源；过期或不可得时明确标注。 |
| 人工核验 | 伤病、停赛、预计首发、临场可用性 | 在提高信心前，检查官方发布会、足协动态、官方首发和可靠球队新闻。 |

`data/source-coverage.json` 是这些层级的工作检查表。预测仍可在存在缺口时发布，但每个未解决缺口都必须降低信心，且不得被当作已确认依据。

## 校准规则

- 小组赛平局地板：热门方胜率低于 60% 时，平局概率通常保持在 28%-32% 附近，除非有强核验证据反向支持。
- 热门方上限：缺少排名、阵容、战术、可用性和市场信号的共同支撑时，不把任何一方推到 60% 以上。
- 零封谨慎：对手存在明确转换、定位球、主场/球迷语境路径时，避免只给 2-0 或 3-0 一条比分路径。
- 数据缺口惩罚：缺少伤病/首发/天气/市场变化时，信心最高为 `medium`；两个以上主要缺口通常应限制为 `medium-low`。
- 反向路径：每份预测必须说明最可能打出非主预测结果的路径。

## 截至第 020 场的经验教训

第 020 场之前的复盘说明，模型仍需要保留第 016 场形成的平局和下风方警惕，但第 017-020 场也说明，强热门一旦把第一波压迫转化成连续机会，比分差距会被拉开。之后每次新增或刷新预测前，都要套用以下经验：

- 紧凑下风方：低位防守加门将发挥，可以把强热门的射门数量压成平局，西班牙 0-0 佛得角就是典型。
- 转换和二点球：澳大利亚 2-0 土耳其、科特迪瓦 1-0 厄瓜多尔、荷兰 2-2 日本都说明，转换执行可以击穿场面优势。
- 定位球和反弹球：沙特 1-1 乌拉圭说明，一次定位球或反弹球就足以打掉零封预测。
- 高温和中断：比利时 1-1 埃及说明，高温、补水暂停和节奏中断会帮助组织性强的下风方。
- 物流和政治背景：伊朗 2-2 新西兰说明，旅行、签证、安保或基地调整会提高丢球和平局风险。
- 临场战术变化：瑞典 5-1 突尼斯说明，近期换帅或阵型大改会让旧防守数据失效。
- 替补影响：比利时扳平和多场后段波动说明，替补影响应绑定到比分情景，而不是只写成泛化风险。
- 热门方比分上限：法国 3-1 塞内加尔、挪威 4-1 伊拉克、阿根廷 3-0 阿尔及利亚、奥地利 3-1 约旦都说明，胜负方向判断正确时，如果热门方有多条独立得分路径，模型仍可能低估两球以上胜出的空间。
- 零封与胜差分离：对手存在转换或定位球威胁时，应降低零封信心，但不应自动压低热门方胜差，尤其当热门方可以通过中路配合、边路传中、定位球和替补深度连续制造机会。
- 后段比分扩张：第 017-020 场说明，即使比赛状态判断正确，如果把替补和落后方冒险只写成泛化风险，也会漏掉 3-1、4-1 或 3-0 的具体路径。

实践调整：从第 021 场开始，只要两项波动变量成立，`保守 / 平局路径` 仍必须是有效替代方案；但当热门方同时具备排名优势、多种得分方式和替补优势时，也要给出可信的胜差扩张路径。需要把「下风方可能进球」和「热门方不能赢两球以上」分开处理。

## 比分情景

从第 017 场开始，每份新增或刷新预测必须给出三种比分情景：

| 情景 | 含义 |
| --- | --- |
| 主情景 | 与公开预测倾向一致的 headline 比分。 |
| 保守 / 平局路径 | 与数据缺口最匹配的小比分或平局路径。 |
| 上限 / 替代路径 | 捕捉主要波动风险的大比分或下风方路径。 |

情景概率是「单个比分」的顶部估计，不是完整概率分布，因此不要求加总为 100%，因为其他比分仍然可能发生。

## 每日总览图

当日报覆盖同一个中国时间日期下两场或更多已预测比赛时，必须在快照之后、单场配图之前加入一张每日预测总览图。

总览图应按每场一行或紧凑区块展示：

- 中国时间开球时间
- 对阵
- 胜平负概率
- 主情景比分及其对应理由
- 保守 / 平局路径比分及其对应理由
- 上限 / 替代路径比分及其对应理由

理由必须来自对应比分情景，不能用一条泛化理由解释整场比赛的所有比分路径。

每份日报必须包含 `Summary Card Notes` / `总览图说明`，说明总览图的阅读方式、使用的证据类型、可能改变预测的临场变量，并包含「不构成任何投资建议」的免责声明。

## 复盘反馈标签

赛后复盘可按需加入一个或多个结构化误差标签：

- `draw_underweighted`
- `favorite_margin_overstated`
- `favorite_margin_understated`
- `clean_sheet_overstated`
- `underdog_resilience_underweighted`
- `transition_risk_underweighted`
- `set_piece_risk_underweighted`
- `data_gap_overconfidence`

这些标签用于指导下一次每日运行的校准，但不回写已经发布的赛前预测。

## 截至第 024 场的经验

- 加纳 1-0 巴拿马验证：当身体对抗和节奏压制是核心变量时，完整的 1-0 小比分路径必须保留。
- 葡萄牙 1-1 民主刚果说明：强热门若仍缺临场首发、天气、市场和下风方直接打法变量，平局路径被低估。
- 英格兰 4-2 克罗地亚、乌兹别克斯坦 1-3 哥伦比亚说明：热门分差可以扩大，同时对手仍能进球；零封概率必须和热门分差上限分开处理。
- 从第 025 场开始，每场预测都保留严肃的 1-1 或下风方替代路径；除非补齐临场球队新闻、天气和市场快照，否则信心不高于中等。
- 当前重点校准标签：`draw_underweighted`、`favorite_margin_understated`、`favorite_margin_overstated`、`clean_sheet_overstated`、`underdog_resilience_underweighted`、`data_gap_overconfidence`。
