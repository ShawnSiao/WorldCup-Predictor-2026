This page stores standing account-introduction copy for World Cup publishing. Use it alongside each match-specific prediction post.

## Copy Rules

- Say clearly that the content is a football match prediction.
- Say that the account uses ChatGPT's highest reasoning model for World Cup match analysis.
- Mention that match forecasts cover win/draw/loss, likely scorelines, and key risk factors.
- Do not present any prediction as guaranteed.
- Include an investment-advice disclaimer where the platform format allows it.
