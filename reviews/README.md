## Review Ratings

- `correct`: the predicted winner or advancement result was correct, and the scoreline or match script was close.
- `partial`: the broad read was useful, but the winner, scoreline, or key match script was partly wrong.
- `wrong`: the prediction missed the core outcome or relied on assumptions that did not hold.
